## Introduction

Desh Videsh Travels is not Just a Another Travel Company. They are a team of Travel Enthusiasts who about Traveling, Exploring New Places, and Make Memories with their Content Creation Ideas. Here, i Joined as a Fullstack Software Developer in there Early Stage. I have Worked on the Frontent and Backend of the Website.

From Redesigning the Website to Implementing New Features, I have Worked on all Aspects of the Website. The Website is Built using NextJS, TailwindCSS, and Typescript. The Website is Using Strapi as CMS for Managing Content Real-time.

Working in a startup environment has been a great learning experience for me. I have acquired new skills and improved existing ones, particularly in SEO optimization, content management, and website redesign. Additionally, I have gained experience with Amazon Web Services (AWS) for managing the website's infrastructure.

## Features Some of the Great Achievements are:

- **Early Stage Development** - Planned and Gather Requirements and Idea's for the Website. Worked on the Initial Design and Development of the Website.

- **SEO Optimization** - Worked on optimizing the website for search engines. Manage Google Analytics ( Google Search Console, Google Analytics, Google API's) and Worked on Improving the Website's Ranking on Search Engines.

- **Implemented Strapi CMS** - Implemented Strapi CMS to the Website for Easily Managing Website Content Real-time.

- **Caching and NextJs** - Implemented Caching in the Website for Faster Loading of the Website. Used NextJs for Server Side Rendering and Implemented Next.js Image for Optimizing Images and Lazy Loading.

- **Framer Motion for Animations** - Implemented Framer Motion for adding animations to the website. Framer Motion is a production-ready motion library for React. With Framer Motion, we can easily add animations to the website and make it more interactive.

- **Blogs and Reels Page** - Implemented Blogs and Reels Page on the Website. The Blogs and Reels Page allows users to read the latest blogs and watch reels related to travel. With the Blogs and Reels Page, users can get information about different travel destinations and plan their trip accordingly.

- **AWS Cost Optimization** - Worked on AWS Cost Optimization for the Website. Managed AWS Services and Infrastructure for the Website, and Worked Regularly on Optimizing the Cost of the Website.

- **Internal Projects and Tools** - Worked on Internal Projects and Developed Tools for Automating Tasks and Managing Operations.
